Plane Extract

The Plane Extract is more an extension of the Join Command. The Plane Extract creates a closed polyline by combining several fragments inclusive of the inner holes/parts. The distinct feature of the Plane Extract is that it does not disturb the original drawing but moves only the selected outline (along with the embedded parts) to the desired location (The Join Tool modifies the original drawing part). So, another plane can then be extracted from it. See the following illustration.

You will now find that has moved the extract to the desired position. The Original drawing part remains untouched.
