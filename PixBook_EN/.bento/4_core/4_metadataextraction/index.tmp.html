For a given 2D model the kernel needs to return a condensed set of meta data information This includes:

  • Part properties (name, external name, material, thickness)

  • Part area, bounding box, center of gravity and weight

  • Length of outer contour

  • Number of inner cutting contours

  • Total length of inner cutting contours

  • Average distance of inner cutting contours

  • Number of inner marking contours

  • Total length of inner marking contours

  • Average distance of all inner contours

  • Number of bends

  • List of all bends (along with the corresponding properties)

  • Number of forms

  • List of all forms (along with the corresponding properties)

  • Number of welds

  • List of all welds (along with the corresponding properties)
