The kernel should automatically add dynamic texts resp. 2D templates (a template is a predefined, persisted 2D model, made of geometry, texts and related attributes eg. like processing type). The following properties should be used by templates:

  • Part name

  • Part number

  • Material

  • Thickness The kernel can either position the text or template directly at a specified position or – depending on the size of the text or template – try to find a best matching position itself automatically. See also Commands_EN.pdf, page 9, "PlaceText” and “RetryPlaceText".
