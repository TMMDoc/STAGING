The kernel can simplify (“smooth”) the given 2D model, by replacing contours with simpler ones. The function allows to define several criteria parameters such as a threshold angle, a distance threshold and a minimal length. Contours are replaced, if a new contour can be calculated, where:

  • corners with more than the threshold angle remain unchanged,

  • the new contour doesn’t deviate more than the distance parameter from the original,

  • no element of the new is shorter than the given minimal length parameter and

  • the new contour has less elements than the original one. See also GeoKernelAPI_20240529.docx, page 16, Smoothing and page 46, ImportSetSmoothMode.
