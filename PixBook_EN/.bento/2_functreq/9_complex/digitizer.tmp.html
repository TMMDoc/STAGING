Digitizer

provides an add-on module that allows it to be used with a digitizer. If your copy of  has this add-on module, it would have been configured for the particular model and make of digitizer you are using. The digitizer's overlay menu will provide button areas that work similar to some of the commands of the drawing editor. For example, you will typically find Line, Polyline, Circle, 3P Circle, Arc, 3P Arc, Point, Zoom and other commands mapped to button areas on your digitizer.

When you are using the digitizer, switches to a digitizer monitor window that tracks the digitizer cursor on-screen. This is handy when you are using 's snaps with the digitizer (you may have to click the Snap button on the digitizer’s overlay menu to enable snaps). When you are finished using the digitizer, you can click the Finish button on the digitizer menu to return back to the drawing editor.
