Parametric Script

can load and run parametric scripts written in the MetaScript language. A script file (typically with a .ps extension) defines a set of parametric shapes. When you load a script file, the script controller shown alongside appears. Each shape defined in the script has a toolbar button at the top. Clicking one of these displays a picture that explains about this shape's parameters. The values for these parameters can be then typed in into the input bar below the drawing. Then, click on the drawing to create the parametric object and position it at the click point.

  • If you point at one of the buttons on the toolbar, a tooltip appears, displaying the name of the shape.

For more information on writing parametric script programs, see Parametric Programming Language.
