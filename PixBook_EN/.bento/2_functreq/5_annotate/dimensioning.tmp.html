Dimensioning

This chapter discusses the dimensioning commands available in . Dimensioning commands are available in the drawing pane, the laser tooling pane and the punch tooling pane. In the drawing pane, you can dimension drawing geometry.

In the laser tooling pane, you can dimension laser tooling entities such as corner loops, approach and escape geometry etc. In the punch tooling pane, you can dimension punch geometry. When you switch between these panes, only the dimensioning that is created in that pane is displayed; dimensioning created in other panes are not displayed. Like all other commands in , dimensioning commands can be undone using the Undo menu item, or redone using the Redo menu item.

Pick

Pick

This mode is used to select dimension objects for deleting or for moving. To select a dimension object, click near the bottom of the text. If the text is centred in the dimension line, click near its centre. If the dimension line is to the right of the text, click on the bottom right of the text. If the dimension line is to the left of the text, click near the bottom left corner of the text. The figure illustrates where to click to select the dimension for different types of dimension (the small circle is the click point where you should click).

A selected dimension can be dragged by clicking on it, holding down the mouse button and moving. The effect of dragging a dimension is different in normal drawing mode, and in dimensioning mode. In drawing mode, the entire dimension moves; this should be used when the dimension has to be moved along with the object is dimensioning.

Therefore, if a dimension is moved in normal drawing mode, it moves as a unit. If it is moved while in the dimensioning mode, only the text point is shifted.

Selecting multiple dimensions

Multiple dimensions can be selected using the Pick tool by holding down the Shift key while clicking on additional dimension objects.

You can also click away from any dimension objects, hold down the mouse and draw a rectangle enclosing the dimensions to be selected.

When you are in the dimensioning mode, the Pick tool will allow you to pick only dimensions, and not any other entities.

Horizontal

Horizontal

This tool is used to create horizontal dimensions. The first two clicks indicate the distance to be dimensioned, and the third click (optional) positions the text.

Holding down the Ctrl key when you click on the second point of the dimension automatically places the dimension without requiring a third click to position the dimension. If the second click is to the right of the first click, the dimension is positioned above the clicked points. If the second click is to the left of the first click, the dimension is placed below the clicked points.

Using custom dimension text

In this mode, as well as in all the other dimensioning modes, the input bar has an input box titled "Text". If you type in any text here, it is used instead of the actual measured dimension. If you leave this blank, measures the actual dimension and inserts it. Note that you can enter any text, not just numbers into the text box.

Using tolerance text

In this mode, as well as in all the other dimensioning modes, the input bar has an input box titled "Tolerance". If you type in any text here, it is added to the dimension as the tolerance. If you input a value, a single tolerance text is added with a plus/minus sign. If you input two tolerance values with | character as a divider between two values, the first tolerance is used with plus sign and the second tolerance is used with minus sign. The input bar has another input box titled "Text". The text you type in this box is displayed along with the tolerance text instead of the measurement value.

Aligning dimensions precisely

The base point of the dimension text acts as a snap point. This can be used to align dimensions precisely. See the example below, which shows how to use snaps to align two horizontal dimensions.

Horizontal Baseline

Horizontal Baseline

This mode is used to create a set of horizontal dimensions that share the same first point. For every additional dimension after the first, you need to supply only the second point and (optionally) the dimension placement point.

Any number of such dimensions can be created using the same first point. To set the first point again, click the Horizontal Baseline toolbar button again.

Horizontal Continue

Horizontal Continue

This mode is used to create a set of horizontal dimensions, like the Horizontal Baseline mode. The difference is that in this mode, the end point of the previous dimension becomes the start point of the next dimension.

Horizontal Ordinate

Horizontal Ordinate

This mode is used to create ordinate dimensions. Ordinate dimensions simply list the distance of a point from some other reference point. The first click in this mode sets the reference point, and every subsequent click creates an ordinate dimension at that point, that displays the horizontal distance between the reference point and the second point.

For clarity, you should not have more than one ordinate reference in your drawing. That is, if you are using ordinate dimensions in your drawing, use one point as the reference point for all the ordinate dimensions.

Vertical, Vertical Baseline, Vertical Continue, Vertical Ordinate

Vertical, Vertical Baseline, Vertical Continue, Vertical Ordinate

These commands work similar to the corresponding horizontal mode tools: Horizontal, Horizontal Baseline, Horizontal Continue, Horizontal Ordinate. Here are some dimensions created using these tools.

Aligned, Aligned Baseline, Aligned Continue

Aligned, Aligned Baseline, Aligned Continue

These commands work similar to the corresponding horizontal mode tools: Horizontal, Horizontal Baseline, Horizontal Continue. They allow creation of dimensions at any angle. The length along the angle between the first and second click points is indicated by the dimension.

Perpendicular, Perpendicular Baseline, Perpendicular Continue

Perpendicular, Perpendicular Baseline, Perpendicular Continue

These modes are similar to the Aligned dimensioning modes, in that they allow dimensions to be created at any angle. The angle in these cases is measured perpendicular from a baseline. The first click in any of these modes selects a baseline (this click must be close to a straight line segment in your drawing), and the subsequent clicks dimension the perpendicular distance between the second point and this reference line.

Smart Dimension

Smart Dimension

Use this single dimension command to add horizontal, vertical or aligned dimensions. The command first starts in aligned mode. When the command is in use, you can switch between horizontal, vertical or aligned modes by pressing the PageUp or PageDown key.

The picture above shows the three dimensions (aligned, horizontal, vertical) that can be placed using this tool. As mentioned earlier, you can switch to these different modes of dimensions using either the PageUp or PageDown key. You can also place horizontal or vertical dimensions using this command. The status bar indicates the type of dimension currently in use.

Radius, Center Radius

Radius, Center Radius

The Radius and Center Point Radius tools are used to dimension radius’s of circles and arcs. The Center Point Radius tool draws the dimension line from the centre of the circle, while the Radius tool does not extend the dimension line to the centre of the circle. By clicking just inside or just outside the circumference of the circle or arc, the dimension can be positioned on the inside, or the outside of the circle or arc.

Like other dimensions, radius dimensions can be repositioned in dimension mode. The dimension leader lines are redrawn to dimension the same circle, with the text at the new position. l Setting the text position to Horizontal in the Dimension Properties dialogue (discussed next) usually leads to better looking radius and diameter dimensions.

Diameter, Center Diameter

Diameter, Center Diameter

These modes work similar to the radius dimensioning modes, but display diameters instead of radius’s. The Center Point Diameter tool draws the dimension line across the centre of the circle to the other end of the diameter, while the Diameter tool does not.

Like other dimensions, diameter dimensions can be repositioned in dimension mode. The dimension leader lines are redrawn to dimension the same circle, with the text at the new position.

Setting the text position to Horizontal in the Dimension Properties dialogue (discussed next) usually leads to better looking radius and diameter dimensions.

Angular

Angular

The Angular dimension tool is used to dimension the angle between two lines.

Hold the shift key for the 3rd click to dimension the obtuse angle.

3-Point Angular

3-Point Angular

The 3-Point Angular dimension tool is used to dimension the angle formed at a point, between two other points. The first click sets the apex point at which the angle is measured; the angle is measured counter-clockwise between the second and third points you click upon.

Hold the shift key for the 4th click to dimension the obtuse angle.

Callout

Callout

This mode is used to add notes to the drawing in the form of callouts. To create a callout, type in the text to be displayed, click to indicate where the arrow should point, and click again to indicate where the text should be placed.

Multi-Dimension Feature

Multi-Dimension Feature

The Multi Dimension tool is designed to allow you to manually dimension geometry with just using a single tool with a single or double click to generate dimensions.

The tool supports 3 modes of dimensioning which can be selected by Right Clicking on your mouse after you have selected the tool.

These are:

Multi Dimension

This will allow you to click on a single entity to get a length or radius, by clicking two entities you can get angular measurements.

Baseline

When you first click the baseline start point will be set and any subsequent dimensions will be taken from that point.

Continue

When you click to dimension the baseline start point will be taken from the point of the last dimension.

Create Bend Profile With Dimensioning

Create Bend Profile With Dimensioning

A New feature to extract bend profiles from a part in 2D CAD or 3D CAD (Sketch Mode). The 'Bend Profile' tool will present the user with options for creating a bend profile which is found under the Dimensioning Tab, this will present a selection of views based on the geometry and Bend Lines in the part from which the user can then click to select the view and place it on the part drawing by moving the mouse and clicking to select the position.

Segment

Segment

This tool is used to add dimensioning for straight line and curved segments. Click on the segment to be dimensioned, and click again to position the dimension. Or, you can hold down the Ctrl key and click on a segment to position the dimension automatically.

Dimensioning all segments of a polyline

If you hold down the Shift key while you click on a polyline, all segments in the polyline are dimensioned automatically; this is equivalent to clicking on each segment of the polyline with this tool to dimension it.

Text

Text

This tool is used to insert descriptive text (without any dimensioning leader lines or arrows). Click on the point where the text is to be inserted, and type in the text into the Create Text dialogue that appears. The text is created at the point where you clicked; the click point can be aligned to the text using three possible horizontal alignments and three possible vertical alignments.

NOTE:

  • You can also create multiple lines of text directly by typing the text accordingly. In the picture shown, the textbox contains two lines. So this text will be displayed as multiline text containing "Line one" in the first line and "Line two" in the second.

  • Text that has been created earlier with this tool can be edited by holding down the Ctrl key and clicking on the existing text.

Edit Text

Edit Text

This tool is used to edit the text of any dimension entity. To edit text, click on the dimension text that has to be edited. The text is now displayed in the Text box in the input bar. Now type in the new text and press Enter. You can also type in the new text and click anywhere on the drawing pane to update the dimension. Tolerance values can also be changed in the same way.

If you have to edit many dimensions,

  • Click on the first dimension text.

  • The dimension text is displayed in the Text box. Type in the new text.

  • Click on the next dimension text to be edited.

  • This updates the first dimension and then loads the second dimension’s text into the Text box for editing.

Dimension Properties

Dimension Properties

Clicking this button brings up the dimension settings dialogue, which allows you to set the arrow head style and size, the font used for dimensions, and the position of the dimension text with respect to the dimension line.

The arrowhead style, arrowhead size, and the dimension font and size are common to all the dimensions in the drawing. When you change any of these settings, all existing dimensions are redrawn using the new settings. However, the text position setting is specific to each dimension, and if you change this setting, the new value is used only for dimensions created afterwards. Existing dimensions are not modified.

  • Turn on the ANSI Standard option to set the height and length of the arrowheads automatically based on the text size.

  • Turn on the Keep trailing zeroes setting to force to keep trailing zeroes after the decimal point. With this setting off, trailing zeroes after the decimal point are removed.

  • Click on the Save as Default button to save the current dimension settings as defaults for all new parts you create.

  • For horizontal dimensions, break-in and horizontal settings produce similar results.

Auto Dimension

Auto Dimension

This tool automatically creates dimensions for the entire part. The dimensions created depend on the options turned on in the AutoDimension Settings dialogue box.

Generate dimensions for: Use this setting to choose whether dimensions are generated for all entities, for selected entities or for unselected entities.

Dimension origin: Dimensions are generated using the point specified here as the reference point. Using the lower left extent option is recommended, since it usually results in the fewest number of dimensions being created.

Dimension style: Use this to choose a dimensioning style. The baseline style is recommended for most drawings, but the ordinate style can be useful in reducing clutter in very complex drawings.

Outside shape: Turn this setting on to dimension irregular polylines, and outer polylines. If this is off, only regular shaped inner holes are dimensioned.

Standard hole positions: Turn this on to dimension the locations of standard shaped holes. For example, in the picture above you can see that the OB 30x10 hole and the CI 10 standard shaped holes' locations are dimensioned.

Standard hole descriptors: Turn this on to add descriptors (as for the obround and circle hole in the picture above) for standard hole shapes.

Standard hole legends: Turn this on to add legends (A, B, C…​) for each kind of regular shaped hole. This can result in less cluttered dimensioning, especially if there are numerous similar holes in the drawing. The View Legends tool can be then used to view the hole chart for the drawing.

Array parameters: If there are patterns of similarly shaped holes (for example, an LAA, BHC, or GRD pattern), this option displays the pattern information. This includes row and column spacing and count for a GRD pattern, or bolt hole diameter and count for a BHC pattern.

Irregular holes: Turn this option on to add segment by segment dimensioning for irregularly shaped holes.

Bend line positions: If you turn this option on, bend line positions are dimensioned (relative to the closest parallel edge of the outer shape).

Arc radius dimensions: Turn this on to annotate the radius of all arc segments.

Delete old dimensions: Turn this option on to delete existing dimensions in the drawing before auto dimensioning. This is recommended, because otherwise the automatically created dimensions could visually interfere with the existing dimensions. You should always run the AutoDimension tool first, and then create any other custom dimensions later.

Zoom

Zoom

This button is used to zoom in and out of a drawing. Here are different ways in which the zoom mode can be used.

  • Type in a zoom factor and press Enter. The centre of the drawing remains unchanged, while the drawing itself is scaled by the factor you enter. Factors less than 1 zoom out, while factors greater than 1, zoom in.

  • Click, hold down the mouse button and drag. This displays a rubber band rectangle on the screen that defines the area of interest and when you release the mouse button that area expands to fill the entire window.

  • Hold down Ctrl and click, to zoom back to your previous view.

  • Hold down Shift and click to zoom back to the extents of the entire drawing.

  • Hold down Alt and click to zoom out by a factor of 0.5

    Other navigation mechanisms

In addition to the zoom command on the drawing editor toolbar, there are some other methods in which you can navigate about the drawing:

  • Pressing Ctrl+E at any time (even halfway through a command) will cause to zoom out so that the entire drawing area is visible.

  • Pressing Ctrl+W at any time (even halfway through a command) will cause to temporarily suspend that command, and switch into a zoom mode. The prompt bar at this time will read "Click, drag and release to zoom in and resume the previous command". After you have clicked, dragged and released to define an area of interest, will zoom in and the command will resume.

  • If you have a mouse with a mouse wheel, you can also use the mouse wheel to zoom and pan. To zoom in, roll the wheel forward, and to zoom out, roll the wheel backward. To pan the drawing without changing the zoom level, click the mouse wheel, and drag the mouse while holding the mouse wheel down. These navigation modes can be used at any time, even when you are halfway through a command.

Click this button to request for a repaint of the screen. This may be sometimes needed to remove blips, or markers, left behind on the screen by an earlier command. Also, when you delete an entity that coincides with another entity, the second entity may not be seen until you click the redraw button.

View Legends

View Legends

This tool is used to view the legends table (hole chart) for the drawing. The legends are added for standard shape holes by the AutoDimension tool, if the Standard hole legend option is turned on.
