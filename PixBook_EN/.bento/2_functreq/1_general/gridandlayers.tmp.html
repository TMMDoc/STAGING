Snap Grid

Snap Grid

provides a snap grid that can be used to align entities to coordinates that are multiples of some round number, during drawing. The Snap Grid command brings up the Grid Properties dialogue, where you can set the grid pitch, and the number of subdivisions in each main division of the grid. For example, the settings alongside (Grid Pitch  10, and Subdivisions  5) will provide one grid point every 2 units.

The Snap to grid checkbox enables the grid; with the grid enabled, coordinates you enter using the mouse are rounded to the nearest grid point. The Display grid checkbox, which can be set on or off independently of the Snap to grid checkbox, lets you choose whether the grid is actually displayed in the drawing window or not. If you choose a grid pitch which is too small in relation to the extents of your drawing, the grid might be too dense to be displayed; in this case, will not display the grid until you zoom closer in, increase the grid pitch, or decrease the number of subdivisions.

The Display Coordinate axes button can be turned on to ask to draw the X and Y axis lines, regardless of whether the snap grid is on or not. If a user coordinate system is now in effect, the rotated, and translated X & Y axes are drawn. (The snap grid itself is never rotated or translated, and is always with reference to the global coordinate system).

Note. The snap grid affects only points entered by clicking with the mouse. Even with the grid turned on, you can still enter points that don’t lie on the grid, by typing them into input boxes that prompt for coordinates.

Layer Setup

Layer Setup

This command displays the Layer Control dialogue, that is used to configure the set of layers in a drawing.

Each layer has a name, a colour, and a line type. The colour and line type can be set by clicking on the appropriate cell, and choosing a new colour or line type from the graphical menu that pops up. In addition, layers can be turned on or off by setting or resetting the Visible flag. Layers can also be Locked to prevent entities on from being edited, or deleted. Entities that are created new are placed on the active layer, which can be set by clicking on Active button of the layer. The Purge command is used to remove empty layers, with no entities in them. (A few layers like 0, Dimension, Bend and MBend are never removed by the Purge command). The Show nodes switch displays small marker blips at each node of polylines on that layer; these can be used to view nodes that may not otherwise be easily seen, such as nodes that lie collinear on a segment.

More operations on layers

Clicking on the More options button displays further operations related to layers.

Move selected entities to this layer: All entities that are currently selected in the drawing are moved to the layer you are currently editing.

Set as default layer configuration: The set of layers is saved in the configuration file, and is used as the set of layers for any new drawings that are created.

Select all entities in this layer: All entities in the drawing that belong to the layer being edited are selected.

Make invisible colours visible: If the background colour of the drawing editor window is same as that of the drawing then they will not be visible. Use this setting to invert the drawing colour so that they are visible in the editor window.

Save by Name: Use this option to save the current layer setup.

Load by Name: Use this option to load previously stored layer setups. When loading a layer set, all the existing layers in the drawing will be removed and new layers from the saved layer set will be loaded. Any existing entities will be moved to these freshly loaded layers.

Special layers

Some layers have special meanings.

For example, you can set up so that all entities in the Marking layer are cut using laser power, frequency, duty cycle and feed rate suitable for marking.
