Fold to 3D Model

The Fold to 3D model command can be used to make a 3D model starting with a developed flat. Here is the procedure for folding a developed flat to a 3D model.

Move all the bend lines to the "Bend" Layer

Select one of the bend lines, by clicking on the line, and select all other lines in the same layer using File/Edit/Select ByLayer. Then, go to the Layer Setup dialogue, click on the Bend Layer, and then choose Move Selected entities to this layer from the More Actions link to move all the selected lines to the Bend layer.

If there are any mountain bends, move those to the MBend layer. The drawing now looks like figure (a), note that there is one bend line in the MBend layer.

Apply bend information to each bend line.

Choose the Edit Bend Information mode. Then, hold down Ctrl and click on the drawing to apply bend information automatically to all bend lines (The CtrlAuto-mark all option). Bend markers are drawn on all the bend lines. The figure (b) above shows the part after applying bend information.

Now, click on the Fold to 3D Model button to fold the model. The resulting model is shown below in the figure.
