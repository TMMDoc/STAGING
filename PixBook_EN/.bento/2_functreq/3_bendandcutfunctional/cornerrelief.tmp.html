  • Corner relief in sheet metal refers to a feature designed to alleviate stress concentrations and prevent material deformation or cracking at inside corners of a sheet metal part.

  • It involves removing a small portion of material, usually in the form of a cut or notch, from the inside corner.

Square Corner Relief

To create square shape profile in the specific corner. Interface

  • Icon for square corner or keyboard short cut. Interactive

  • Select square corner relief icon.

  • By clicking the corner user should be able to create (square or rectangular) shape profile at the 45 degrees angle. 

  • Modify width and height parameters.

Obround Corner Relief

Create obround shape profile in the specific corner. Interface

  • Icon for obround corner or keyboard short cut. Interactive

  • Select obround corner relief icon.

  • By clicking in the corner point user should be able to create (obround) shape profile at the 45 degrees angle. 

  • Modify width height and radius parameters. 

V-Notch Corner Relief

Create V shape profile in the specific corner. Interface

  • Icon for v shape corner or keyboard short cut. Interactive

  • By clicking the corner point, user should be able to create (V) shape profile at the 45 degrees of angle.

  • Width, height and radius parameters can be modified. 

Circular Corner Relief

Create circular shape profile in the specific corner. Interface

  • Icon for circular corner or keyboard short cut. Interactive

  • Select circular corner relief icon.

  • By clicking in the corner point user should be able create (circular) shape profile at the 45 degrees of angle. Enable modifying width, height and radius parameters.

  •  Able to modify width and height and diameter parameters.
