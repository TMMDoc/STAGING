Edit Bend Information

This tool is used to edit the bend information that is attached to a bend line. The bend information gets attached to a bend line in one of these ways:

  • When a 3D-model is created in the 3D modeller and unfolded, automatically attaches bend information to each bend line in the unfolded drawing. (This works only if you unfold the model using true thickness; it does not work if you unfold the model as a paper model).

  • When you use the Plane Joining or Flange Joining tools to create developments, bend information is computed and attached to the bend lines that created.

  • When part drawing is imported from a DXF format, and there is an associated bend data file in BND format, imports the bend information and attaches it to the bend lines as they are created.

When you switch to this mode, displays small markers on each bend line that has associated bend information. The marker visually indicates if the bend line is a mountain bend or valley bend. Clicking on such a marked line brings up the bend information inspector. Various attributes of the bend information can be edited in this inspector. Converting a normal line to a bend line

If you click on a line that is not a bend line, confirms if you want to convert it to a bend line, and then attaches bend information to it, and brings it up for editing.
