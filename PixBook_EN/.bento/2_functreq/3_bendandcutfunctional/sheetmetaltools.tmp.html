Fillet Tool

Create rounded corners between two lines. User Interface

  • Icon for fillet tool or keyboard short cut. Interaction

  • Select (inside or outside) fillet tool icon.

  • Input the desired values for radius R.

  • Choose the edge where you want to create the appropriate fillet.

Chamfer Tool

Create bevelled edges between two lines. User Interface

  • Icon for chamfer tool or keyboard short cut

  • Parameters: XY distance and angle Interaction

  • Select inside chamfer tool icon.

  • Input the desired values for distance 1 (x) and distance 2 (y) and angle.

  • Click on the edge where you want to create the chamfer.

Offset Tool

Create parallel copies of lines, polylines, or curves at a specified distance. User Interface

  • Icon for offset tool or keyboard short cut Interaction

  • To retain the original polyline, hold down Ctrl while confirming the offset.

  • To offset only a single segment, hold down Shift.

  • If the offset direction is incorrect, undo and enter a negative offset distance.

Copy notch

To copy notch property and use it latter on selected place. User Interface

  • Icon for notch copy or keyboard short cut Interaction

  • The Copy Corner Notch tool allows you to copy a notch from one corner to another.  

 

Delete notch.

To delete notch property. User Interface

  • Icon for notch copy or keyboard short cut Interaction

  • This tool removes notches at corners or along line segments. 

  • Click on adjacent line segments, and the notch is deleted.  


Mirror notch

To mirror notch property based on reference line. User Interface

  • Icon for mirror notch or keyboard short cut. Interaction 

  • Click on adjacent line segments, and the notch is mirrored.  

  • For edge mirroring, it’s mirrored about an imaginary line passing through the notch’s midpoint and perpendicular to the edge.  

  • For corner mirroring, it’s mirrored about a line through the corner and bisecting the angle.
