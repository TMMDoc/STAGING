This chapter desribes the user interface functionality of CAD software.

Entity types

  • Polyline (lines + arcs), Point, Text, Spline,

  • Dimension, Bend-line, Block

Entity creation commands

  • Lines: Line, Connected Lines, Parallel Line

  • Arcs: Center-Point-Arc, 2-Point, 3-Point, Tangent

  • Rectangles: Corner-Rectangle, Center-Recentangle

  • Circles: Center-Point-Circle, 2-Point Circle, 3-Point Circle, 2-Tangent Circle, 3-Tangent Circle

  • Polygon: Inscribed, Circumscribed, Edge

  • Text: Simple annotation type text using vector fonts (similar to SHP files)

  • Dimensions: Simple, Baseline, Continue, Ordinate (Horizontal / Vertical), Radius, Diameter

  • Other: Point
