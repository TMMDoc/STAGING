PIX is a light weighted CAD sketch tool which supports our sheet metal customer with specific funcitonalit. It is not meant to replace a full blown CAD solution.

  • The system should have an intuitive graphical user interface (GUI) for creating, editing, and viewing designs.

  • The tool should have 2D drafting capabilities, including lines, circles, arcs, polylines, and text annotations.

  • The system should have a layer-based drawing system to manage design elements efficiently.

  • The system should have transformation tools (move, rotate, scale, mirror, array).

  • The tool should allow undo/redo functionality for safe editing.

  • The system should support constraints and relations (e.g., perpendicularity, parallelism) for accurate modifications.

  • The tool should have an object snapping and grid system for precision.

  • The system should support scripting for automating repetitive tasks.

  • The tool should have a strong API integrations to extend functionality.
