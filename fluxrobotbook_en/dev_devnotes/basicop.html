Basics on the Robot

Robot is in "normal R&D" operation in T1 mode (reduced speed). If we bend with 100% then it needs to be in T2 mode

To start up the system

  • remove the E-stop

  • set the views to 3 views (1=prog, 2=Usercoord, 3=Joint coord)

  • load a program on RA and the Robot

  • start the program

reset errors

hold the deadman switch in the back

if the fault LED is on press > reset <

jog

>SHIFT< key and blue axis key

in which coord you choose by >COORD< key

override must not be 0% (upper right corner)

select programs

>Select< button

>enter<

Run a program

press the shift and FWD button

With the step button you could do a step wise run

Views

With >view< you rotate trough the windows, with shift-view the number of windows could be selected With > POSN < the coordinate could be selected

export a program

select the drive via Menue > Files > Util > set device > UT1:

Press the print button, enter to confirm the name

user frames

*check user frames → should be all in the default state (see frames)
