Postprocessor Matlab

This chapter describes the principles of the current postprocessor which creates the Fanuc .LS program

Matlab implementation

Variables

Variable array:

MCS

all way points in Joint coordinates

WCS

all way points in Cartesian coordinates

sMCS sWCS

points for the ram motion

seq

Variable for sequence (see below)

  • currently a variable seq decides for each step if

    • colum 1: in which User frame is it taken

    • colum 2: joint=1 lin movement=2

    • colum 3: Joitcoord=1 Cart=2

Programm

  • a standard header is copied

  • For each point the post prozessor looks into the folder \PLC (this example e.g. point 9).

    • if a file "pre_9.txt" is existent it copies the content of this file in front of the motion command

    • the the motion command is written

    • if a file "post_9.txt" is existent it copies the content of this file after the motion command

  • Afterwareds all point information is written to the file
