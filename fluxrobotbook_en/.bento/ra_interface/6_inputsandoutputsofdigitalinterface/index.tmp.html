6. Inputs and outputs of digital interface

Robot to machine

Release condition for press beam downward

movements

To release a downward movement of the press beam, the safety signal XS7:17-18 should also be bridged by the robot. The signal XS8:3 however can be reset individually by the robot to functionally stop the press beam.

Releasing robot axes and no error active

To be able to carry out a bending step, the signal XS8:4 "Robot OK" must be set on the robot and remain set for the duration of the bend. Otherwise the downward movement of the press beam stops immediately. This input should be used to stop the downward movement of the press beam immediately in the case of a robot error, especially when the robot is supporting the workpiece during the bending operation.

XS8:5

Robot to machine

Positioning backgauge

On a positive edge of the signal, backgauge, lower tool movement and crowning are positioned. The exact point in time at which these axes are positioned depends on how the change of the bending steps has been programmed on the Human Machine Interface.

XS8:6

Robot to machine

Release from robot to approach mute point

The signal must remain set during the complete movement to the mute point, otherwise the press beam will stop immediately. The robot should only reset the signal once the press beam is located at the MP (XS7:3 TRUE).

XS8:7

Robot to machine

Release from robot to approach clamping point

The signal must remain set during the complete movement to the clamping point, otherwise the press beam will stop immediately. The robot should only reset the signal once the press beam is located at the CP (XS7:4 TRUE).

XS8:8

Robot to machine

Release from robot to approach lower dead point

The signal must remain set during the complete movement to the lower dead point, otherwise the press beam will stop immediately. The robot should only reset the signal once the press beam is located at the lower dead point (XS7:5 TRUE).

XS8:9

Robot to machine

Release from robot to approach upper dead point

The robot should only reset the signal once the press beam is located at the upper dead point (XS7:1 TRUE).

XS8:10

Robot to machine

Error acknowledgement, starting bending program, selecting first bending step and starting

On a positive edge of the signal, the bending program is restarted from the first bending step, the output XS7:12 "First bend active" is set. If necessary, the hydraulics are switched on fully automatically before this, and the tools are clamped. If an error is present on the machine, the edge of the signal first carries out an error acknowledgement, the bending program is then restarted. The robot can reset the signal once the machine has started. If the safety fence is closed (for safety-related and functional reasons) and the machine is not yet referenced, referencing is started with the edge of the signal (XS7:12 TRUE).

XS8:11

Robot to machine

Stopping bending program and hydraulics off

Robot → Press

On a positive edge of the signal, the bending program is stopped, and the hydraulics are switched off. The signal should be reset by the robot, since otherwise no automatic program start is possible anymore (set for 2 seconds).

Robot to machine

Tool change

On a positive edge of the signal, the machine readies itself for the tool change (see "Robot as tool changer", pg. 10‐33). If ready, the machine sets the signal XS7:29 TRUE. The robot should reset the signal to start the bending program.

XS7:5

Machine to robot

Press beam at the lower dead point

The output is set by the machine once the press beam is located at the lower dead point, and is reset on leaving this point.

XS7:6

Machine to robot

ACB programmed and in use

The output is set by the machine at the lower dead point if Automatically Controlled Bending (ACB) with identification during bending step was programmed. The output is reset at the decompression point after the post bend.

XS7:7

Machine to robot

ACB identification bend active

The output is set by the machine at the lower dead point if ACB with identification during bending step was programmed.

After the identification bend, the press beam moves to the decompression point to release the sheet.

The output is reset on reaching the lower dead point during post bending.

XS7:8

XS7:9

Tilting

XS7:8 Positive tilting

XS7:9 Negative tilting

The output is only set by the machine if the press beam is located at the decompression point, in case the robot must compensate for the tilting of the sheet.

XS7:10

Machine to robot

Bend finished

The output is set by the machine only once the sheet has been bent. If ACB with identification is programmed, the output is set at the end of the second bend.

XS7:11

Machine to robot

Machine without error and robot activated

The output is always set when no errors are present on the machine. If the bending program is interrupted due to an error on the machine, then this output is reset. This signal can be used to stop the robot movement.

XS7:12

Machine to robot

First bend active

The output is set if the bending program is in the first bend. If the bending step is advanced or the program is stopped, then this output is reset. With a manual program this signal is always set, since by definition the manual program is always in the first bend.

Robot to machine

2-channel safety release of the press beam. The 2 channels must be bridged or opened at the same time.

XS7:17-18

Contact circuit 1

XS7:19-20

Contact circuit 2

A pulse signal from the safety controller of the machine is used on the pins 17 and 19. Both contacts are bridged by the robot if the press beam may be released. Pins 18 and 20 are connected back to the machine’s safety control.

To release a downward movement of the press beam, the functional signal XS8:3 is set by the robot.

XS7:21-22

Robot to machine

Robot operating mode selection for rapid downward movement (K)

Rapid downward movement of the press beam without optical protective device. The functional signal XS8:2 should also be set by the robot.

XS8:2 and XS7:21-22 must always be set or reset at the same time by the robot, otherwise an error will occur on the machine.
