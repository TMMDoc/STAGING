The RightAngle Robot Interface provides you a all tools to integrate a robot to a TruBend1000 series bending machine. the recommmendet solution is a decentralised setup which works by a direct integration of the components to a device panel which is directly connected to the the Fanuc switch cabinet.

The data exchange between the machine and robot contoller works with both:

  • a simple 42 pin digital interface

  • or via EtherCAT communication

Types of integration

Standalone operation

In compliance with the safety regulations of the relevant national standards and laws as well as the additional safety information in this description, it is also possible to operate the TruBend as usual via the operating panel with manual loading. We differentiate between the operations "TruBend without robot" and "TruBend with robot".

When operating without a robot, the TruBend is loaded manually. For this purpose, the press brake is controlled with the operating panel. This setup status corresponds to the description in the operator’s manual of the press brake.

TruBend with robot

During operation with a robot, the sheet is guided by the robot.

If you select the operating mode "TruBend with robot", the foot switch is partially inactive, e.g. the "PRESS BEAM DOWN" foot switch has no function. Therefore, in robot mode it is not possible to trigger a movement of the press beam with the foot switch.

Operating-mode key switch As standard the external operating-mode key switch is used to switch between operation with and without robot. The operating-mode key switch is located next to the On/Off switch.

CAUTION

The robot is started up while the operator is located within the danger zone of the robot!

Make sure that a parking position has been set up for the robot for stand-alone operation.
