Design

A good gripper is often the key to a good bend sequence in FluxBend. In order to find an efficient solution you might take minigripper.STL and modify the coordinates of the suction cups until you have a good solution. Example material is in 04_components\Gripper. The STL file (binary) represents the collision geometry, the dat file the position of the geometrical properties, information about the suction cup position, group and type. List of all available suction cups can be found in the next page

The content of the .dat file is:

NAME=Custom
CUPNAME=SAF50
CUPZPOS=186
CUPS=4
CUP1POS=100,85 #1
CUP2POS=100,-85 #2
CUP3POS=200,85 #3
CUP4POS=200,-85 #4

Import the Gripper via Edit > Database > Bend Grippers

After finding the optimal position you could produce a Gripper.

Note
 The Gripper height in the assembled state needs to be measured in the activated state (sucktion on).
