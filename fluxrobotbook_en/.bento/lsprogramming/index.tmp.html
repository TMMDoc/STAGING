Flux outputs a .rnc file with all information needed for all subsequent programs. A separate postprocessor will convert the output to your specific system. Major components are

  • The main file used on the robot (Programname.ls)

  • Sub programs for motion sequences (PickupCentering.ls)

  • Sub programs to position the part before a bend (BendXPositioning_Sub.LS)

  • Sub programs to bend a part (BENDXSUB.LS)

  • Files needed on RA control (BEND3_RamPts.txt)
