There are two basic motion commands we use

J P[10:J_Aloft] 100% FINE ; "J"= Joint movement "10:J_Aloft" = Point 10, the J_ in front of the point name indicates it is in joint coordinates "100% FINE" = joint motions have a speed in percent

L P[13:C_Mute Point] 2000mm/sec FINE "L"= Linear movement "13:C_Mute Point" = Point 13, the C_ in front of the point name indicates it is in cartesian coordinates "2000mm/sec FINE" = joint motions haare in mm/sec

The format of the points is quite intuitive

P[1:"J_init"]{

GP1:
UF : 1, UT : 2,
J1 =     90.00  deg,	J2 =    0.00  deg,	J3 =     -20.00  deg,
J4 =     0.00  deg,	J5 =   20.00 deg,	J6 =     0.00 deg
};

P[4:"C_Height"]{
