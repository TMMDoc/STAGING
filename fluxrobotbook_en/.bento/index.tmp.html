FluxRobot is Metamation’s next-generation offline programming system. It automatically provides all NC code for the Robot and your bend machine.

FluxRobot is fully configurable but it is advised that you follow the processes and wiring of this manual. If done so FluxRobot provides a easy access to the functionality of

  • Pickup from a stack (tilted and straight) - with a vacuum gripper

  • Pickup from a dispenser

  • Part separation with a cross blow

  • Double sheet detection

  • Centring Table

  • Bend programm generation

  • Drop parts

Both with - Vacuum grippers - Pinch grippers

Moreover FluxRobot provides macros for - Part stack management - Tool management
