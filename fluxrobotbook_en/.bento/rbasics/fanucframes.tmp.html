User frames

FluxRobot on your desktop PC should be as close as possible be the digital representation of your real system. It is recommended to update the data of your PC with the machine regularly to get the best results to avoid collisions. FluxRobot provides tools and procedures to do this in a guided way.

Nevertheless FluxRobot gives the operator on the machine to do fine adjustments, especially for effects like sheets sagging on the gripper by gravity etc.

In FluxRobot / Fanuc the number of frames are limited so only the crucial components work in a user frame. The less crucial components need to be adjusted in FluxRobot accordingly. The default usage of FluxRobot uses the following frames:

  • Frame 1: Robot (main frame)

  • Frame 2: Pickup Station 1 (e.g Dispenser)

  • Frame 3: Pickup Station 2 (e.g. Pallet)

  • Frame 4: Centering Table

  • Frame 5: Double sheet detector

  • Frame 6: Tool changer

If a operator needs to fine adjust a frame to run a program then he could tweak a frame within some milimeter. Generating the next Flux program this component positons need to be corrected in Flux that the next output could be again done with identical frames
