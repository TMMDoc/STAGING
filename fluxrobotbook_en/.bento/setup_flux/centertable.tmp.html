The Centering table provides a accurate part gripping position in case the pallet is inaccurate.

How to enable Centering table:

Open a program in Flux.

Click on Add â€“ Cell component

Component glob will open and choose the Simple Centering Table. Set the values in following order to positioning the Centering table (Refer the attached image):

Angle= 90
Tilt X Angle= 0
Tilt X Angle= 38
X,Z position as -1400, -792
Lift= 800

The above values are calculated based on setup we are using in our shop floor. This process will enable the Centering table in Flux. Refer the image

How to Use Centering Table:

Open pickup glob: click on the P in bend navigator

Select Centering table check box. It will use the centering table

Now the part will be placed over the centering table for re-positioning before proceeding for bending.

→ The centering table will get a part detection sensor in the table plate (prep electrical wiring!)
