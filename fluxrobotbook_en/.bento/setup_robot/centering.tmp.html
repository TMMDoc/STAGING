Calibration of the centering table

A calibration plate is used with markings in 50mm distance. The markings are nubered 1,1 .. 6,6. The calibration routine in 03_Matlab_Ccode\ML_Calibrationprogs generates a fanuc program which needs to be executed

The calibrated probe tool is used.

Note
 all frames should be set to the values of FRAME 1 (machine - robot frame)

The in the ideal case the tip of the probe is centered on the cross and the gauge shows exactly 5mm. If the component is not correctly the exact position of the cross needs to be touched up and the matlab calibration program calculates the new offset for Flux
