To get the location of the components up to 1mm a simple pointer is sufficient. The simple pointer tool is used.

Pointer

Note
 if the tool frame 1 and Userframe 1 is not zero the calculation will not work !

  1. select a tool frame 1 set to zero

  2. touch a vertial sheet from both sides an note down the wrist coordinate in world coordinates

  3. the length of tye pointer is (X2-X1-sheet thickness ) / 2

  4. enter the calculated length in Toolframe 1 and check if the pointer rotates around a point

Calibration of the the measurement gauge

for the precise insert of the part into the machine a precise pickup is needed. This is either the tilted dispenser, the tilted pallet pickup or the centering table. Best practise is to use a mechanical gauge which could be calibrated to a middle position.
