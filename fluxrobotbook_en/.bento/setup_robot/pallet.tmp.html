Coarse position of the pallets

All components are measured with the ponter. The centering table and the regrip station are lateron fine adjusted with the dial indicator.

The pallets are set to the coordinates in World (Robot coordinates ) [-1200 -1500 Z] and [-0 -1500 Z] The Z-Value is measured with the pointer

For Flux we need to provide the coordinates in UserFrame 1; Since the pallet is referenced to the X positon of the robo rail this value need to be considered. Finally we need to have all parameters adjusted that the pallet corner shows with the measurement tool the right coordinates
