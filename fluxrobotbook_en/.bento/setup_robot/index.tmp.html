Home position

A good home position is with J = [90 0 -20 0 20 0], then you are not in singularities

Tool frames

FluxRobot uses different tools which have their specific tool frame in the Fanuc control

  • Straight tools like pointer and measuring gauges

  • Pinch grippers

  • Vacuum grippers

  • Tool changer

User frames

All components are configurable in their position in FluxRobot. Components which are critical for the accuracy of the parts have their own user frame on the fanuc control that they could be fine calibrated without the offline programming system.

Measuring the User Frames

  1. Press Menu.

  2. Press Setup.

  3. Select Frames.

  4. Press enter key.

  5. Press Other.

  6. Select User frame.

  7. Select the desired Frame.

  1. SelectThree Point.

  2. SelectOrient Origin Point.

  3. Move to the zero point of the desired frame.

  4. Press shift key + Record.

  5. Move the measuring tip toward the X-axis. This measurement is used to teach the coordinate system here its X-alignment is.

  6. SelectX Direction Point.

  7. Press shift key + Record.

  8. Move the measuring tip toward the Y-axis. This measurement is used to teach the coordinate system here its Y-alignment is.

  9. Press shift key + Record. The measured data is saved automatically
